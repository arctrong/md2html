Introduction

Integral tests check the final results of the modules execution. They are the same for all implementations.

Implementation

The test framework Python unittest is used. Beautiful Soup library is used for HTML parsing.

Prerequisites

To run the integral tests Python 3 is required. The following additional Python package must be installed:

$ python3 -m pip install beautifulsoup4

All tested modules must be built (if required). All unit tests must be passed.

Execution

>cd %MD2HTML_HOME%\test

>test.bat

or 

$ cd $MD2HTML_HOME/test

$ ./test

The result must look like this:

===== py implementation integral tests =====
.........................
----------------------------------------------------------------------
Ran 25 tests in 7.420s

OK

===== java implementation integral tests =====
.........................
----------------------------------------------------------------------
