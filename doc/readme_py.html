Usage

A simple usage example is:

>python %MD2HTML_HOME%/python/md2html.py -i test.txt

This will convert file test.txt into file test.html using default parameters. For other options use argument -h or run the module without arguments.

Development

There are no special packaging actions required. The file python/md2html.py is a self-contained executable module.

To run tests execute:

>%MD2HTML_HOME%\python\test.bat

The output must end with something like this:

.................................................................
----------------------------------------------------------------------
Ran 65 tests in 0.037s

OK

Arguments may be used like this test.bat -v that will give a verbose output. Run python -m unittest discover --help for all possible parameters.
